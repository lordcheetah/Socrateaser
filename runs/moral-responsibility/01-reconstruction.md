# Basic-Desert Moral Responsibility — Reconstruction
> **Run:** moral-responsibility · **Mode:** full · **Stage:** Reconstruct
> **Thesis:** No one is ever morally responsible in the basic-desert sense, because basic desert requires being the ultimate source of one's action, and no one is ever the ultimate source of what they do.

---

## Standard form

P1. If an agent is morally responsible for an action in the basic-desert sense,
    then that agent is the ultimate source of that action.
    *(Conceptual premise: basic desert requires ultimate sourcehood.)*

P2. No agent is ever the ultimate source of any action they perform.
    *(Metaphysical premise: ultimate sourcehood is never instantiated.)*

C.  Therefore, no agent is ever morally responsible for any action in the
    basic-desert sense.

## Validity

The argument is **valid**. It is a straightforward universally-quantified modus
tollens (denying the consequent):

- P1 has the form: ∀x∀a (BasicDesertResponsible(x, a) → UltimateSource(x, a)).
- P2 has the form: ∀x∀a ¬UltimateSource(x, a).
- From any instance — BasicDesertResponsible(x, a) → UltimateSource(x, a) and
  ¬UltimateSource(x, a) — we infer ¬BasicDesertResponsible(x, a) by modus tollens.
- Universally generalizing over agents x and actions a yields C: ∀x∀a
  ¬BasicDesertResponsible(x, a).

No suppressed premise is needed to get from P1 and P2 to C. The conditional in P1
runs in exactly the direction the inference requires: sourcehood is stated as
*necessary* for basic desert, and P2 denies that necessary condition obtains, so
basic desert is denied. (Had P1 instead asserted sourcehood as *sufficient* for
basic desert — "if ultimate source, then responsible" — the argument would commit
denying-the-antecedent and be invalid. It does not; the framed thesis says basic
desert "*requires*" ultimate sourcehood, which is the necessary-condition reading.
This is worth flagging only so the council attacks the right reading.)

Two clean-up notes on the inference, neither of which threatens validity:

1. **Scope of "action."** P2 must range over the same domain as P1 — every action
   for which an agent might be a candidate for basic desert. The framed thesis
   asserts P2 universally ("anything they do"), so the quantifiers match and the
   generalization to C is licensed.
2. **No equivocation on "ultimate source."** Validity holds only if "ultimate
   source" carries the *same* sense in P1 and P2. The thesis pins one sense (the
   buck-stopping, regress-ending originator — not the compatibilist
   reasons-responsive sense). As long as both premises use that single pinned
   sense, the term does not equivocate and the inference goes through. Whether each
   premise is *true* under that single sense is a soundness question, not a
   validity one, and is left to the council.

## Where the weight sits

**P1 — the conceptual premise — carries the real weight**, with P2 a close and
independently contestable second.

The reason P1 is the load-bearing premise: it is the premise that every
*compatibilist* opponent (the largest opposing camp) denies while conceding
everything else. Source/reasons-responsiveness compatibilists (Fischer–Ravizza,
Frankfurt, Sartorio) and Strawsonians (Wallace, Hieronymi) grant P2 outright —
they happily agree no one is an ultimate, regress-ending source — yet reject the
conclusion. They can only do so by denying P1: by holding that basic desert does
*not* require ultimate sourcehood (compatibilists) or does not require any
metaphysical sourcehood at all (Strawsonians). If P1 falls, P2 becomes harmless: a
true premise about a condition that desert never needed. So P1 is the hinge on
which the entire skeptical argument turns against the modal opponent.

P2 is where the *libertarian* fight happens (Kane, O'Connor concede P1 and attack
P2 via agent-causation). It is genuinely contestable and is the site of the
manipulation/luck arguments. But notice the asymmetry: P2 is defended by an
argument (the regress: any action traces to character, which traces to prior
causes the agent did not produce; neither determinism nor indeterminism breaks
this). That defense is strong and widely granted. P1, by contrast, is an analytic
claim about what "deserve... for its own sake" *presupposes*, and it is granted by
*no* opposing camp — both compatibilists and Strawsonians reject it. The premise
denied by the most opponents, and the one whose truth is least independently
argued, is P1. That is the likely crux.

## Charitable variants

The standard form above is already the charitable reconstruction (it uses the
necessary-condition reading of "requires," avoiding the invalid sufficiency
reading). Three further refinements strengthen it against predictable attacks:

**Variant A — Disambiguate "ultimate source" to forestall the equivocation
charge.** Opponents will say the skeptic trades on an impossibly strong sense of
"source" in P2 and then imports a weaker sense into P1. The charitable fix is to
fix one sense explicitly in both premises:

- P1′. If an agent basic-desert-deserves blame/praise for an action, then the
  agent is the *buck-stopping originator* of that action — the explanation of the
  action does not bottom out wholly in factors the agent did not themselves
  produce.
- P2′. No agent is ever the buck-stopping originator of any action (every action's
  explanation bottoms out in constitutive luck, prior causes, or chance — factors
  the agent did not produce).

This is the version most resistant to the equivocation objection, because the
single italicized definite description is reused verbatim.

**Variant B — Make P2 a conclusion of a sub-argument (Strawson's regress), so the
metaphysical premise is itself supported rather than asserted.** Charitably, the
author need not assert P2 flat-footed; they can derive it:

- P2a. To be the ultimate source of an action, one must be (at least in part) the
  source of the self / character / set of reasons from which one acts.
- P2b. To be the source of one's own character, one must have chosen it on the
  basis of some prior character or set of principles.
- P2c. That prior character was either inherited (not self-produced) or chosen on
  the basis of a still-prior character — launching an infinite regress that cannot
  be completed.
- P2d. Indeterministic "breaks" in the chain add only chance, not authorship, and
  chance is not a form of sourcehood.
- ∴ P2. No agent is ever the ultimate source of any action.

This is the Basic-Argument backbone. It does not change the top-level validity; it
shows the skeptic *why* P2 is supposed to hold, and relocates the metaphysical
dispute to P2a–P2d (where the libertarian denies P2c/P2d).

**Variant C — Retreat to a conditional / restricted conclusion if both universal
premises are judged too strong.** If the council finds P1 or P2 indefensible as
stated, the strongest fallback the author could mean is:

- C′. *Insofar as* moral responsibility is understood in the basic-desert sense,
  and *if* ultimate sourcehood is required for it, then no one has it.

This is weaker (it conditionalizes on P1 rather than asserting it) but preserves
the sharp point — that nothing *short of* regress-ending sourcehood can underwrite
desert-for-its-own-sake. It concedes the crux to the compatibilist while keeping
the skeptical thesis intact against anyone who grants P1. The framer's universal
formulation is the stronger and intended claim; C′ is the disciplined retreat
position, not the primary reading.

---

**Hand-off to the hidden-premise-excavator:** No premise is needed to *repair
validity* (the argument is already valid). But two assumptions are doing silent
work and should be excavated: (i) the **univocity assumption** — that "ultimate
source" means the same thing in P1 and P2 (if it shifts, the argument equivocates
rather than turning invalid); and (ii) the assumption embedded in P1 that basic
desert is a *coherent and non-empty* concept in the first place — a premise the
argument presupposes rather than states, and one a deflationist could attack from a
different direction than the compatibilist.
